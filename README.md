## Setting up Jetstream 

Step-1. Run this command

"composer require laravel/jetstream"

Step-2

"php artisan jetstream:install livewire"

Step-3

"npm install"
and
"npm run dev"

Step-4

"php artisan migrate"

## Install Socialite Package

Run this Command

"composer require laravel/socialite"
